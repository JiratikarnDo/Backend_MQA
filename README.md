# Backend_MQA

Backend ระบบ MQA เขียนด้วย **FastAPI + SQLAlchemy + MySQL**

---
## Features
- ล็อกอินด้วย Google
- ส่งเอกสารเกี่ยวข้องกับ มคอ3 / มคอ5
- ระบบแจ้งเตือน
- จัดการข้อมูลภายในมหาวิทยาลัย
- ฯลฯ
